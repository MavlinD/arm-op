# Сервис выдачи паспортов качества потребителям

### Build Setup:

``` bash
# Download repository:
git clone http://oais01.azot.kmr:8300/mdv/arm-op

# Go to the app:
cd arm-op

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8080/
npm run dev

# Output will be at dist/ folder
npm run build
```
